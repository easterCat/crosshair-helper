📦 Windows应用打包说明

🚀 快速打包
双击运行 build.bat 文件即可自动打包

📋 打包步骤
1. 确保已安装 Node.js 和 npm
2. 双击 build.bat 或手动执行以下命令：
   - npm install (安装依赖)
   - npm run build-win (构建Windows安装包)

📁 输出文件
- 打包完成后，exe文件将生成在 dist 目录中
- 文件名格式：屏幕准星助手 Setup 1.0.0.exe

⚙️ 打包配置
- 安装程序：NSIS格式
- 可选择安装目录
- 自动创建桌面快捷方式
- 自动创建开始菜单快捷方式

🎯 功能特性
- 200款预设准星
- 实时样式切换
- 透明度、尺寸、位置调节
- 热键控制 (F6/F7)
- 覆盖模式

📋 系统要求
- Windows 10/11
- 无需额外依赖

🔧 故障排除
如果打包失败：
1. 确保所有依赖已正确安装
2. 检查 main-fixed.js 文件是否存在
3. 确保有足够的磁盘空间

打包完成后即可分发安装文件！
